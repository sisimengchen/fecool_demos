# fecool

a collection of simple demos of fecool

host 配置 127.0.0.1 feresource.com

## 开发使用


### 开发环境环境部署
```shell
git clone https://github.com/sisimengchen/fecool_demos.git
cd fecool_demos
npm install
```

### 开发环境环境构建
```shell
npm run dev
```

### 开发环境监听文件变化构建
```shell
npm run watch
```

### 启用开发服务器（需要打开新终端）
```shell
npm run server
```

## 生产环境构建
```shell
npm run build
```
